# BLE 协议分析

## 设备信息

### 服务 UUIDs

| UUID 类型 | UUID |
|-----------|------|
| 主服务 | `0000fff0-0000-1000-8000-00805f9b34fb` |
| 写入特征 | `0000fff1-0000-1000-8000-00805f9b34fb` |
| 通知特征 | `0000fff2-0000-1000-8000-00805f9b34fb` |

### 设备名称

常见的设备名称:
- `DG-LAB`
- `Coyote`
- `DG-LAB Coyote`

## 通信流程

### 连接流程

1. 扫描并发现设备
2. 连接到设备
3. 发现服务和特征
4. 订阅通知特征
5. 发送初始化命令
6. 开始正常通信

### 心跳

- 间隔: 5 秒
- 超时: 30 秒无响应则断开

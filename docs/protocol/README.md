# 协议文档

本目录包含 DG-LAB 设备通信协议的分析文档。

## 目录

- [BLE 协议](./ble.md) - 蓝牙低功耗通信协议
- [WiFi 协议](./wifi.md) - WiFi 通信协议
- [数据包格式](./packets.md) - 数据包结构详解

## 协议分析工具

推荐使用以下工具进行协议分析：

- **Wireshark**: 网络数据包分析
- **nRF Sniffer for Bluetooth LE**: BLE 数据包捕获
- **Android HCI Snoop Log**: Android 设备蓝牙日志

## 注意事项

⚠️ 协议文档基于逆向分析，可能不完整或不准确，请以实际设备行为为准。
